MML-Multicorr is an extension and web application implementation of Multicorr, a correlation pattern hypothesis test program developed by Steiger (1979). MML-Multicorr may be cited in APA style as follows:

Fouladi, R. T, & Serafini, P. E. (2018). MML-Multicorr. Retrieved from http://members.psyc.sfu.ca/labs/mml.

Offline and bootstrap versions of MML-Multicorr are available at its github page.
